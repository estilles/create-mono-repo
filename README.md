# create-mono-repo
Home for a number of quick configuration utilies, to be used in the form of ...

```bash
npm [init/create] @estilles/...
```

## create-esm
Initialize a Node/ESM repository

Usage:

```bash
# use with npm init
npm init @estilles/esm
```
```bash
# use with npm create
npm create @estilles/esm
```
```bash
# use with npx
npx @estilles/create-esm
```
